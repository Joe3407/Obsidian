# Motion Basics

## Key Concepts
- **Scalar:** Magnitude only (distance, speed)
- **Vector:** Magnitude + direction (displacement, velocity, acceleration)

### Comparisons
| Quantity     | Type   | Real-life Example           | Notes                  |
| ------------ | ------ | --------------------------- | ---------------------- |
| Distance     | Scalar | Running 3 km around a track | Always positive        |
| Displacement | Vector | 3 km north from start point | Can be negative / zero |
| Speed        | Scalar | Fan spinning at 100 RPM     | Only magnitude         |
| Velocity     | Vector | Car moving north at 60 km/h | Magnitude + direction  |

---

## Equations

### Equations of Motion (Uniform Acceleration)
- $v = u + at$  
- $s = ut + 0.5at^2$  
- $v^2 = u^2 + 2as$  
- $X_f - X_i = \frac{1}{2} (V_i + V_f) t$


### Average Velocity & Acceleration
- $v_{\text{avg}} = \Delta s / \Delta t \ (\text{or } (s_2 - s_1)/(t_2 - t_1))$  
- $a_{\text{avg}} = \Delta v / \Delta t \ (\text{or } (v_2 - v_1)/(t_2 - t_1))$


---

## Special Cases
### Velocity
- **v = 0** in the following cases:  
  1. Distance/displacement = 0 (object hasn’t moved)  
  2. Instant when changing direction (e.g., ball at the top of its upward throw)

- **Uniform Velocity:**  
  - v ≠ 0 but constant in magnitude and direction  
  - a = 0  

### Acceleration
- **a = 0** in the following cases:  
  1. Velocity is uniform (magnitude and direction constant)  
  2. Velocity = 0 and object is at rest (not moving)
